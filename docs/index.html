Chesslike board examples

Knight's tour

The motivation behind Chesslike Board is presenting puzzles on a non-standard chess board. Here is a knight's tour on a 3x7 board. 

const { createBoard, movePiece, setPiece } = ChesslikeBoard;

const board = createBoard({
	columns: 3,
	rows: 7,
	target: '#knights-tour',
	view: { border: true },
});

const moves =
	'c2 b4 c6 a7 b5 c7 a6 c5 b7 a5 b3 c1 a2 c3 b1 a3 c4 b2 a4 b6'.split(' ');
while (true) {
	// Put a white knight in a1.
	await setPiece(board, 'a1', 'wn');
	// This doesn't change anything, just pauses for the length of a move.
	await movePiece(board, 'a1', 'a1');
	let lastSquare = 'a1';
	for (const newSquare of moves) {
		await movePiece(board, lastSquare, newSquare);
		lastSquare = newSquare;
	}
	// Another pause for the length of a move.
	await movePiece(board, lastSquare, lastSquare);
	// Remove the knight from its finishing position before starting again.
	await setPiece(board, lastSquare, null);
}

You can create more than one view of a board. This view is linked to the same board and has the rows and columns transposed. This view also uses pieces from a different set. 

const { createView } = ChesslikeBoard;

createView(board, {
	target: '#knights-tour-alt',
	transform: 'transpose',
	pieces: 'fa7',
});

Pieces, notation and positions

Pieces can be placed on the board using the notation bk for black king, wn for white knight etc. Alternative sets have disc (bd and wd) and circle (bc and wc) icons. There is no theoretical limit to the size of a board, so for example setPiece(board, 'cv150', 'wb') will place a white bishop in the top right square of a 100x150 board. You can also place and move pieces using [column, row] notation e.g. setPiece(board, [100, 150], 'wb').

Transformations

Probably the most useful transformation is 'rotate' which displays the board from the other player's point of view. The 'transpose' transformation is illustrated above in the knight's tour example and is particularly useful for responsive displays when working with non-square boards. All possible transformations are supported and are demonstrated below, along with the code to generate them.

Please note that the display of row and column labels in the border does not currently work with transformations: see #1.

const { createBoard, createView } = ChesslikeBoard;

const transforms = 'rotate transpose flipV flipH rotateR rotateL flipD';
const position = 'rnb/3/RNB';
const board = createBoard({ rows: 3, columns: 3, target: '#base', position });
for (const transform of transforms.split(' ')) {
	createView(board, {
		target: `#${transform}`,
		transform: transform,
	});
}

Fool's mate

Chesslike Board can be used to show chess games, although if this is what you want to do other packages offer more features.

The views from both white and black's point of view below play out the "fool's mate" game. 

const { createBoard, createView, movePiece, setPosition, clear } = ChesslikeBoard;

const board = createBoard({ target: '#fools-mate' });

createView(board, { target: '#fools-mate-black', transform: 'rotate' });

while (true) {
	await setPosition(board, true);
	await wait(2000);
	await movePiece(board, 'f2', 'f3');
	await movePiece(board, 'e7', 'e6');
	await movePiece(board, 'g2', 'g4');
	await movePiece(board, 'd8', 'h4');
	await wait(2000);
	await clear(board);
}
